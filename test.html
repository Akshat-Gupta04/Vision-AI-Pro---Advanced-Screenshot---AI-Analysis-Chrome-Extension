"Could not establish connection" Error:

  • This happens when the content script isn't loaded yet
  • Solution: Refresh the page and wait 2-3 seconds before using the extension
  • The extension now automatically injects the content script if needed
  • Make sure you're not on a chrome:// or extension page

Continue Buttons Hidden/Cropped:

  • The dialog now automatically adjusts its size to fit your screen
  • Continue buttons will always be visible at the bottom
  • If content is long, you can scroll within the dialog
  • Look for the "‚Üì Scroll for more" indicator if content overflows

Extension Not Working:

  • Check if the extension is enabled in Chrome Extensions (chrome://extensions/)
  • Make sure you have a valid API key entered
  • Ensure you're on a regular webpage (not chrome:// or extension pages)
  • Try reloading the extension from the Chrome Extensions page
